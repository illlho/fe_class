[html 문서 태그 정리]
- 기본 태그
<!doctype html> : 문서의 형식을 정의
<html> : html 문서의 시작과 끝을 정의
<meta> : 문서의 정보를 정의
<head> : 문서의 정보를 정의
<title> : 문서의 제목을 정의
<body> : 문서의 내용을 정의

- 텍스트 태그
<h1> : 책 제목 / 문서에서 가장 중요한 제목 (한 페이지에 보통 1개만 사용)
<h2> : 챕터 제목 / 주요 섹션(큰 주제)의 제목
<h3> : 소제목 / 챕터 안의 세부 항목
<h4> : 세부 소제목 / <h3>보다 더 작은 단위의 제목
<h5> : 세부 항목의 부제목 / 거의 사용되지 않지만, 필요하면 활용 가능
<h6> : 가장 작은 소제목 / 거의 사용되지 않으며, 내용 정리를 위한 보조 제목

- 목록 태그
<ul> : 순서가 없는 목록을 정의
<ol> : 순서가 있는 목록을 정의
<li> : 목록 항목을 정의

- 링크 태그
<a> : 하이퍼링크를 정의

- 이미지 태그
<img> : 이미지를 삽입

- 이미지 + 설명 태그
<figure> : 이미지, 그래프, 표 등을 독립적인 콘텐츠 블록으로 묶음
<figcaption> : <figure> 내부 콘텐츠의 설명을 추가

- 이미지 맵 태그 (링크 영역 지정)
<map> : 클릭 가능한 영역을 설정하는 맵을 정의
<area> : 맵 내에서 클릭 가능한 특정 영역을 지정

------

위는 간단하게 html 문서의 기본적인 태그를 정리해 봄.
추가적으로 설명할 것은 사용했을 때 어떠한 이점이 있기 때문에 종류별로 태그를 나눠 사용하는지,
그리고 태그를 사용할 때 어떠한 점을 주의해야 하는지에 대해 설명할 예정.
또한 예제를 통해 실제로 어떻게 사용하는지 학습하고 과제를 통해 직접 만들어보며 익히는 과정을 가질 예정
