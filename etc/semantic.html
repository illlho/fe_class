HTML 시맨틱 태그 설명

시맨틱 태그는 문서의 구조와 내용에 의미를 부여하는 HTML 요소입니다. 이 페이지에서는 HTML5에서 제공하는 모든 시맨틱 태그의 사용 예시를 보여드립니다.

구조적 태그 (Structural Tags)

<header> - 헤더

페이지나 섹션의 머리말 부분을 정의합니다. 주로 제목, 로고, 네비게이션 등을 포함합니다.

회사 소개

2010년 설립된 혁신적인 기술 기업

<nav> - 내비게이션

웹 페이지 내 주요 내비게이션 링크 모음을 정의합니다.

  • 홈
  • 제품
  • 서비스
  • 연락처

<main> - 메인 콘텐츠

문서의 주요 콘텐츠를 정의합니다. 페이지당 하나만 사용할 수 있습니다.

웹 개발의 기초

텍스트 관련 태그 (Text-level Tags)

<time> - 날짜/시간

<figure> 및 <figcaption> - 그림과 설명

독립적인 콘텐츠(이미지, 다이어그램, 코드 등)와 그에 대한 설명을 정의합니다.

<mark> - 강조 표시

참조나 표시 목적으로 하이라이트된 텍스트를 정의합니다.

작품의 제목 등 인용의 출처를 정의합니다.
