프론트엔드 강의

웹 개발의 기초부터 고급까지!

강의 주차

1주차: HTML 기초

HTML의 기본 문법을 배우고 구조를 이해합니다.

2주차: CSS 기초

CSS를 이용하여 웹 페이지의 스타일을 적용하는 방법을 배웁니다.

3주차: Flexbox와 Grid

Flexbox와 Grid를 이용한 레이아웃 기법을 익힙니다.

4주차: JavaScript 기본

JavaScript의 기본 문법과 변수를 학습합니다.

5주차: DOM 조작

JavaScript를 이용한 DOM 조작 방법을 배우고, 웹 페이지와 상호작용할 수 있습니다.

6주차: 이벤트 처리

웹 페이지에서 사용자 이벤트를 처리하는 방법을 배웁니다.

7주차: ES6 문법

ES6에서 추가된 새로운 문법을 소개하고 사용법을 익힙니다.

8주차: 비동기 처리

JavaScript에서 비동기 처리 방법을 배우고, Promise와 async/await를 사용합니다.

9주차: API 사용

외부 API를 호출하고 데이터를 처리하는 방법을 익힙니다.

10주차: 프론트엔드 최적화

웹 페이지의 성능을 최적화하는 방법을 배우고 적용합니다.
