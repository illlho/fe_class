JavaScript 코드 구조의 기본 요소

JavaScript 코드는 다음과 같은 기본 요소로 구성됩니다:

  • 문장(Statement): 작업을 수행하는 코드 블록
  • 세미콜론(;): 문장의 끝을 나타냄
  • 블록(Block): 중괄호 {}로 둘러싸인 코드 그룹
  • 주석(Comments): 코드를 설명하기 위한 텍스트

문장(Statement)

JavaScript 문장은 코드에서 하나의 명령을 나타냅니다. 각 문장은 일반적으로 세미콜론(;)으로 끝납니다.

// 각각의 줄이 하나의 문장입니다
let message = "안녕하세요";
console.log(message);
alert(message);

세미콜론(;)

세미콜론은 JavaScript 문장의 끝을 나타냅니다. 대부분의 경우 자동 세미콜론 삽입(ASI) 기능 때문에 생략할 수 있지만, 명시적으로 사용하는 것이 권장됩니다.

세미콜론 생략의 위험성

자동 세미콜론 삽입 기능은 종종 예상치 못한 결과를 가져올 수 있습니다. 아래 예제를 살펴보세요:

// 세미콜론이 없을 때 문제가 발생하는 예
let a = 5
let b = a
(function() {
  console.log("이 함수는 즉시 호출됩니다")
})()

// 위 코드는 다음과 같이 해석됩니다:
// let a = 5;
// let b = a(function() { console.log("이 함수는 즉시 호출됩니다") })();
// 이는 'a is not a function' 오류를 발생시킵니다.

세미콜론 사용 예제

// 권장: 모든 문장 끝에 세미콜론 사용
let name = "홍길동";
let age = 25;
console.log(`이름: ${name}, 나이: ${age}`);

블록(Block)

중괄호 {}로 묶인 코드 그룹을 블록이라고 합니다. 블록은 여러 문장을 그룹화하는 데 사용되며, 조건문, 반복문, 함수 등에서 사용됩니다.

// 조건문에서의 블록 사용
if (true) {
    // 이 부분이 하나의 블록입니다
    let message = "조건이 참입니다";
    console.log(message);
}

// 함수에서의 블록 사용
function greet() {
    // 이 부분이 함수 본문 블록입니다
    let greeting = "안녕하세요";
    return greeting;
}

주석(Comments)

주석은 코드를 설명하는 텍스트로, 실행되지 않습니다. JavaScript에서는 한 줄 주석(//)과 여러 줄 주석(/* */)을 사용할 수 있습니다. 

// 이것은 한 줄 주석입니다
let x = 10; // 변수 x에 10을 할당합니다

/*
  이것은
  여러 줄
  주석입니다
*/
let y = 20;

효과적인 주석 작성 팁

  • 왜(why)에 집중: 코드가 무엇을 하는지(what)는 코드 자체가 설명하고, 주석은 왜 그렇게 했는지를 설명합니다.
  • 복잡한 로직 설명: 이해하기 어려운 코드 부분에 주석을 추가합니다.
  • 불필요한 주석 지양: 코드가 자명한 경우 중복 설명은 피합니다.
  • 주석 최신 상태 유지: 코드를 수정할 때 관련 주석도 함께 업데이트합니다.

코드 포맷팅

가독성 좋은 코드를 작성하기 위한 포맷팅 규칙입니다:

들여쓰기(Indentation)

// 들여쓰기를 사용한 예제
function calculateArea(width, height) {
    if (width > 0 && height > 0) {
        let area = width * height;
        return area;
    } else {
        return 0;
    }
}

중괄호 스타일

// K&R 스타일 (JavaScript에서 많이 사용)
if (condition) {
    // 코드
} else {
    // 코드
}

// Allman 스타일
if (condition) 
{
    // 코드
} 
else 
{
    // 코드
}

한 줄 코드

// 짧은 if 문을 한 줄에 쓰는 경우
if (x > 0) return x; // 한 줄일 때는 가능

// 더 긴 코드는 여러 줄로 나누는 것이 좋습니다
if (longCondition && 
    anotherCondition) {
    // 코드
}

요약

  • 문장(Statement): JavaScript의 기본 실행 단위로, 세미콜론으로 끝납니다.
  • 세미콜론(;): 문장의 끝을 표시하며, 명시적으로 사용하는 것이 권장됩니다.
  • 블록(Block): 중괄호로 묶인 코드 그룹으로, 여러 문장을 하나의 단위로 만듭니다.
  • 주석(Comments): 코드를 설명하는 텍스트로, 실행되지 않습니다.
  • 코드 포맷팅: 가독성을 높이기 위한 일관된 코드 작성 규칙을 적용하는 것이 중요합니다.
